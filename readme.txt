A useless plugin that honks like a goose every time you press the "space" key
 
== Description ==
 
If you want to prank your editors with some goose honk noises, this plugin will do the job!

This plugin will "honk" every time somebody presses the "space" key.
 
== Installation ==
 
This section describes how to install the plugin: 

1. Upload the directory `honking-goose` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
